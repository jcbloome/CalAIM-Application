# **App Name**: CalAIM Pathfinder

## Core Features:

- User-Friendly Interface: Multi-step guide to simplify the CalAIM application process, presenting information in digestible sections.
- Personalized Dashboard: Dashboard for users to view in-progress and completed applications, with status tracking and editing capabilities.
- CS Member Summary Form: Multi-step form to collect necessary member information, divided into logical sections.
- Pathway Determination: Dynamically determine the correct CalAIM pathway (SNF Transition vs. SNF Diversion) based on user input.
- Form Status Tracking: Track the status of each required form for a specific pathway, indicating whether it's pending or completed.
- Admin Interface: Admin section for staff to manage all applications, filter by status, and request revisions.
- AI Form Guidance Tool: A tool to analyze submitted information and suggest relevant next steps or highlight missing information in the forms, helping users complete the process accurately and efficiently. The tool can also prioritize which fields in each form need attention and flag when the application requires manual intervention.

## Style Guidelines:

- Primary color: Professional blue (#4782D1), aligning with the design requirement to use (hsl(220, 60%, 50%)) for primary buttons, links, and key headers, promoting trust and stability.
- Background color: Very light gray (#F0F0F0), as defined, for a clean interface.
- Accent color: Encouraging soft orange (#E68A36) for form section highlights and progress indicators, which aligns with the requested (hsl(25, 80%, 55%)).
- Font: 'Inter' sans-serif, noted as suitable for all text (headings and body).
- Consistent usage of icons from ShadCN UI to create a cohesive design, especially for progress tracking and form status indicators.
- Card-based layout with rounded corners and subtle shadows for a modern, layered look, as indicated. Fully responsive design for mobile-friendliness.
- Subtle animations when transitioning between steps or submitting forms to provide a smooth user experience.
